# FitJourney 💪

Персональное PWA для тренировок и похудения — путь с **126 кг до 80 кг** за 12 месяцев.
Тёмный «стеклянный» интерфейс, оптимизированный под iPhone и установку на экран «Домой».

## Возможности

- **Годовая программа** — 156 тренировок (Пн/Ср/Пт), 6 фаз с безопасной прогрессией и разгрузочными неделями.
- **Только «незаметные» упражнения** — тренажёры, гантели и блоки, работа сидя/стоя на одной станции. Без упражнений на полу.
- **Акцент на кардио** — ротация беговой дорожки, велотренажёра и лестницы + интервалы с 6-го месяца.
- **Выполнение тренировки** — пошагово, с логированием подходов, таймером отдыха и заменой упражнений.
- **Отслеживание веса** — график факт/план, прогноз достижения цели, история с дельтами.
- **Календарь, история, статистика, достижения, серии (streak).**
- **Библиотека упражнений** с техникой, советами, поиском и избранным.
- **Автосохранение**, экспорт/импорт данных, офлайн-режим (Service Worker).

## Стек

Next.js 15 (App Router) · TypeScript · TailwindCSS · Prisma + SQLite · Framer Motion · Recharts · собственная cookie/JWT-авторизация.

## Запуск

```bash
npm install            # зависимости + prisma generate
npm run db:push        # создать SQLite-базу по схеме
npm run db:seed        # заполнить: пользователь, упражнения, программа
npm run dev            # разработка → http://localhost:3000
```

Продакшн:

```bash
npm run build
npm run start
```

## Вход

Учётные данные единственного пользователя задаются в `.env`:

```
APP_USERNAME="r1soX"
APP_PASSWORD="..."
AUTH_SECRET="длинный-случайный-ключ"
DATABASE_URL="file:./dev.db"
```

> Смени `AUTH_SECRET` на свой случайный ключ в продакшене. Пароль можно поменять в Настройках.

## Полезные команды

| Команда | Описание |
|---|---|
| `npm run db:seed` | Заполнить/обновить программу и упражнения |
| `npm run db:reset` | Пересоздать базу и заполнить заново |
| `npm run db:studio` | Prisma Studio (просмотр данных) |
| `node scripts/generate-icons.mjs` | Перегенерировать PWA-иконки |

## Структура

```
prisma/
  schema.prisma        # модели данных
  seed.ts              # заполнение БД
src/
  app/
    (app)/             # защищённые страницы + нижняя навигация
    login/             # экран входа
    api/auth/          # login / logout
  components/          # UI и фичи (workout, weight, calendar ...)
  lib/
    program/           # генератор программы, упражнения, график похудения, достижения
    data.ts            # выборки и агрегаты (дашборд, статистика, серии)
    actions.ts         # server actions (мутации)
    auth.ts / session.ts
public/                # manifest, sw.js, иконки
```

Программа генерируется кодом (`src/lib/program/generator.ts`) и засеивается в БД — легко менять фазы, упражнения и прогрессию в одном месте.
